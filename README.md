# PyGalaxian

![](https://github.com/shivamshekhar/PyGalaxian/raw/master/Screenshots/1.png)

### 描述：

一款簡單的垂直太空射擊遊戲

### 技術：

使用 Python 2.7 和 pygame 庫構建

### 版本和發布：

首個版本，版本 1.0

### 目標平台：

Windows/Linux/Android（未來將支援）

### 說明與前提條件：

運行此遊戲：

* 確保您已安裝 Python 2.7 和 pygame (http://www.pygame.org/) 函式庫

* 運行 PyGalaxian.py

##### 控制：

* 按下左或右鍵移動飛船

* 按上鍵發射雷射

* 消滅所有敵人即可贏得遊戲

* 隨時按 ESC 鍵退出遊戲
